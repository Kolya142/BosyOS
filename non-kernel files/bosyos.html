usage:
$!V(V - hexdigit) - set foreground in vga color
$*V(V - hexdigit) - set background in vga color
\$ - write $ sign
\\ - write \ symbol
\n - write newline symbol
$U - move up for one symbol
$D - move down for one symbol
$L - move left for one symbol
$R - move right for one symbol
